## ลง golang ในเครื่องจากลิ้งค์

- https://go.dev/dl/ เลือก OS ให้ตรงกับเครื่อง
- เปิด cmd แล้วพิมพ์ go version เพื่อเช็คว่าลง go แล้ว

## run project ด้วยคำสั่ง

- go run main.go

## run test

- go test ./...

## มีตรงไหนใช้ AI ช่วยบ้าง

- ช่วยสร้าง logic คำนวณจากโจทย์ใน overheat.go
- ช่วยเขียน test สำหรับทำ unit test
- ช่วยวางโตรงสร้าง handler, repository(DB), model, router, config
- ช่วยแยก package ต่างๆออกจาก main ให้เรียบร้อย (จากโค้ดที่ AI กองไว้ที่ main.go)
